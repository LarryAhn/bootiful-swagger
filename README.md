# Bootiful Swagger
- Using Swagger Demo Project for Spring Boot

## Run Application
```bash
// git clone
$ git clone https://github.com/LawrenceAhn/bootiful-swagger.git && cd bootiful-swagger
$ ./mvnw spring-boot:run

# Running application
http://localhost:8080/swagger-ui.html
```
